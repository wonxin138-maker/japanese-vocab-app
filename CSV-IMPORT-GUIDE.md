# 日语词库导入功能指南

## 功能介绍

在 App 顶部的导入按钮可以上传 CSV 文件，快速添加自己的日语词汇到学习库中。

## CSV 文件格式

### 正确格式

```csv
word,reading,meaning
食べる,たべる,吃
飲む,のむ,喝
行く,いく,去
```

**必须满足条件：**
- 第一行是表头：`word,reading,meaning`
- 每行用逗号 (`,`) 分隔三个字段
- 字段顺序必须是：日文、假名、中文释义
- 使用 UTF-8 编码

### 字段说明

| 字段 | 说明 | 示例 |
|------|------|------|
| word | 日文单词（汉字或假名） | 食べる、猫、日本 |
| reading | 假名读音（平假名或片假名） | たべる、ねこ、にほん |
| meaning | 中文释义 | 吃、猫、日本 |

## 示例词库

使用项目根目录的 `sample-vocabulary.csv` 作为参考模板。

```csv
word,reading,meaning
食べる,たべる,吃
飲む,のむ,喝
行く,いく,去
来る,くる,来
見る,みる,看
```

## 使用步骤

1. **准备 CSV 文件**
   - 用记事本或 Excel 创建 CSV 文件
   - 第一行输入表头：`word,reading,meaning`
   - 从第二行开始输入词汇

2. **点击导入按钮**
   - 在 App 中点击"📥 导入日语词库"按钮
   - 选择你的 CSV 文件

3. **自动处理**
   - 文件上传后自动读取和解析
   - 系统会验证数据格式
   - 导入成功后显示"✅ 导入成功！已添加 X 个单词"

4. **立即学习**
   - 导入的词汇会立即添加到学库
   - 可以在"Feed"页面看到新单词
   - 就可以开始学习了！

## 错误处理

### 常见错误和解决方案

| 错误提示 | 原因 | 解决方案 |
|---------|------|---------|
| ❌ 文件为空 | CSV 文件没有内容 | 检查文件是否有数据 |
| ❌ CSV 格式错误 | 缺少表头或数据行 | 确保有表头和至少一行数据 |
| ❌ CSV 中没有找到有效的数据行 | 所有数据行都缺少必要字段 | 检查每行是否有三个字段 |
| ❌ CSV 解析失败 | 文件编码或格式问题 | 保存为 UTF-8 编码的 CSV |

## 数据支持

✅ **完全支持**
- 日文汉字（水、火、木、等）
- 平假名（ひらがな）
- 片假名（カタカナ）
- 中文（简体、繁体）
- 空格和特殊字符会被自动清理

❌ **不支持**
- 使用制表符 `\t` 或其他分隔符（必须用逗号）
- 没有 UTF-8 编码的文件（可能出现乱码）

## 最佳实践

### 1. 分类学习

为不同主题创建不同的 CSV 文件：
```
动词.csv      - 各种动词
名词.csv      - 常见名词
形容词.csv    - 形容词
```

### 2. 逐步导入

- 不要一次性导入太多词汇
- 建议每次导入 20-50 个单词
- 这样可以更好地消化和学习

### 3. 检查数据质量

导入前检查 CSV 文件：
- ✅ 确保没有空行
- ✅ 确保每行都有三个字段
- ✅ 检查假名是否正确
- ✅ 中文释义要准确

## 数据持久化

所有导入的词汇都会：
- 保存到浏览器的 LocalStorage 中
- 刷新页面后仍然存在
- 不会上传到云端（完全本地）

## 导出数据

当前版本不支持导出功能，但数据完全存储在本地，可以：
1. 按 F12 打开浏览器开发者工具
2. 进入 Application → LocalStorage
3. 找到 `allVocabulary` 键
4. 复制其内容即可备份

## 注意事项

⚠️ **重要**
- 导入后不能撤销，请备份原始数据
- 清除浏览器数据会删除所有词汇（包括已导入的）
- 不同浏览器的数据不会同步

## 技术细节

### CSV 解析逻辑

```javascript
1. 用 FileReader 读取文件（UTF-8 编码）
2. 按 \n 分割行，过滤空行
3. 跳过第一行（表头）
4. 每行用 , 分割成三个字段
5. 验证三个字段都不为空
6. 转换为词汇对象并添加到库
7. 保存到 localStorage
```

### 验证规则

- 文件不能为空（size > 0）
- 至少有表头 + 1 行数据
- 每个数据行必须有 3 个字段（word, reading, meaning）
- 三个字段都不能为空

## 常见问题

**Q: 可以重复导入相同的词汇吗？**
A: 可以，系统会添加新词汇，不会自动去重。建议导入前检查是否已存在。

**Q: 导入的词汇会影响原有功能吗？**
A: 不会，导入的词汇与原有功能完全兼容。

**Q: 文件编码一定要是 UTF-8 吗？**
A: 是的，为了正确显示日文字符，必须使用 UTF-8 编码。

**Q: 可以导入多少个词汇？**
A: 理论上无限制（受浏览器 LocalStorage 限制，通常 5-10MB）。

**Q: 导入后如何编辑词汇？**
A: 当前版本不支持直接编辑，可以重新导入或清空数据重新开始。
