### 2️⃣ 自己创建 CSV 测试文件

**方法 1: 使用记事本**
```
1. 打开记事本 (Notepad)
2. 输入以下内容:
```csv
word,reading,meaning
勉強する,べんきょうする,学习
旅行する,りょこうする,旅行
友達,ともだち,朋友
```
3. 保存为 test.csv (确保编码为 **UTF-8**)
4. 导入到应用
```

**方法 2: 使用 Excel**
```
1. 创建三列:
   A. word      → 食べる
   B. reading   → たべる  
   C. meaning   → 吃

2. 第一行设置表头: word, reading, meaning
3. 从第二行开始输入数据
4. 另存为 CSV (UTF-8)
5. 导入到应用
```

## 故障排除

### 问题: 导入显示成功但词汇没出现

**原因**:
1. 浏览器缓存问题
2. 词汇 ID 冲突

**解决**:
```
1. 硬刷新: Ctrl + Shift + R
2. 清除 localStorage: 
   localStorage.clear()
3. 重新导入
```

---

### 问题: 日文字显示为方块 (乱码)

**原因**:
- CSV 文件编码不是 UTF-8

**解决**:
```
1. 用 VS Code 打开 CSV 文件
2. 右下角检查编码
3. 如果不是 UTF-8，点击选择 UTF-8
4. 重新保存
5. 重新导入
```

---

### 问题: 导入后 App 崩溃

**原因**:
- CSV 格式严重错误
- 数据超过 localStorage 限制

**解决**:
```
1. 打开 DevTools Console
2. 检查错误信息
3. 验证 CSV 格式
4. 清空 localStorage: localStorage.clear()
5. 刷新页面重新开始
```
