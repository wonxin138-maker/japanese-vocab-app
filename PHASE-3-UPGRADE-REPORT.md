### 4. 情感检测增强
**文件：pet-chat.js**

#### 新增方法

##### `_detectStress(text)` 
- **功能**：检测文本中的压力关键词
- **返回值**：压力级别（0=无, 1=轻微, 2=中等, 3=严重）
- **逻辑**：关键词+1，最多3级
```javascript
// 例：用户输入"我好累，太烦了"
// 匹配：'累'(1) + '烦'(1) = 2级中等压力
```

##### `_getComfortResponse(text, stressLevel)`
- **功能**：根据压力类型返回定制化安慰
- **实现**：
  1. 逐个检测 `tiredness` → `overwhelm` → `sadness` 关键词
  2. 优先匹配具体类型
  3. 降级到级别匹配
  4. 最后返回通用安慰

#### 对话流程优化
```
用户输入 → 问候/告别 → 【压力检测】← 新增！
         → 爱意 → 生气 → 开心 → ...
```
**优先级**：问候 > 告别 > **压力检测** > 其他情感

### 动画时序
```javascript
// 眨眼动画细节
@keyframes cat-blink {
  0%, 10%, 20%, 100% { height: 35px; }          // 睁眼
  5%, 15% { height: 4px; }                       // 闭眼
}
// 结果：3秒周期内，5秒时闭眼，15秒时再闭眼
```

### 压力检测算法
```javascript
_detectStress(text) {
  let count = 0;
  for (keyword of stressKeywords) {
    if (text.includes(keyword)) count++;
  }
  // 0关键词 → 级别0（无压力）
  // 1关键词 → 级别1（轻微）
  // 2关键词 → 级别2（中等）
  // 3+关键词 → 级别3（严重）
}
```
