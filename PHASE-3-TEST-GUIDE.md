### 测试2：压力关键词检测 💬

**场景A：轻微压力**

```

```

**场景E：无压力**

**步骤**：

**预期结果**：
- [ ] 有压力关键词 → 安慰短语
- [ ] 无压力关键词 → 普通回复

## ⚠️ 常见问题排查

### Q1：猫咪不显示
**检查**：
```javascript
// F12 Console 输入：
document.querySelector('.cat-3d')
// 应该返回 DOM 元素，不是 null
```

**解决**：
- 检查 pet.html 中是否有 `<div class="cat-3d">` 
- 检查 styles.css 是否正确引入

### Q2：动画不流畅
**检查**：
```javascript
// 关闭其他标签页
// 检查 GPU 加速是否启用
```

**解决**：
- 硬件加速可能未启用
- 尝试其他浏览器

### Q3：安慰短语不显示
**检查**：
```javascript
// F12 Console 输入：
PET_DATA.stressResponses
// 应该看到 comfort, tiredness, overwhelm 等
```

**解决**：
- 清除浏览器缓存（Ctrl+Shift+Delete）
- 重新加载页面（Ctrl+F5）

### Q4：点击没反应
**检查**：
```javascript
// F12 Console 输入：
petState.mood
// 应该返回 0-100 的数值
```

**解决**：
- 检查 JavaScript 是否启用
- 查看 Console 中是否有错误
