# 🚀 AI宠物 - 快速开始

## ⚡ 60秒快速体验

### 步骤1: 打开应用
在浏览器中打开 `pet.html` 文件（Firefox/Chrome/Safari都支持）

### 步骤2: 开始互动
- 💬 **在下方输入框输入文本，按Enter发送**
- 🖱️ **点击左侧粉色宠物玩耍**
- 🎁 **点击"惊喜"按钮获取彩蛋**

### 步骤3: 了解状态条
左上角的三条状态条显示：
- 💕 **心情（粉色）**: 越满代表宠物越开心
- 🍖 **饥饿（橙色）**: 越满代表越饿
- ⚡ **精力（蓝色）**: 越满代表精力越充足

## 🐛 常见问题速答

### Q: 我的数据会丢失吗？
**A:** 不会！数据保存在浏览器本地，关闭页面也不会丢。除非清空浏览器缓存或用`localStorage.clear()`。

### Q: 能不能指定宠物的回复？
**A:** 目前还不能，但代码设计允许轻松扩展。未来版本支持！

### Q: 为什么有时宠物不说话？
**A:** 
- 刚启动时可能需要5秒加载
- 检查浏览器控制台(F12)是否有错误
- 尝试刷新页面

### Q: 宠物会不会累？最终会"老"吗？
**A:** "累"会（精力值会下降），但目前没有衰老系统。未来版本可能加入！

### Q: 怎样导出对话记录？
**A:** 在控制台运行：
```javascript
const history = chatAI.getHistory()
const json = JSON.stringify(history)
copy(json)  // 复制到剪贴板
```

## 🎨 自定义修改教程

### 修改宠物名字
打开 `pet-data.js`
```javascript
// 第3行
name: '毛毛',  // 改成你想要的名字
```

### 修改宠物颜色
打开 `pet.html`，找到SVG部分
```html
<!-- 身体 -->
<circle cx="100" cy="120" r="45" class="pet-body" fill="#FFD6E8"/>
<!-- 改成你喜欢的颜色 #ABC123 -->
```

### 添加自己的对话
打开 `pet-data.js`，找到 `responses` 对象
```javascript
greeting: [
    '你的新对白1',
    '你的新对白2',
    // 添加更多...
]
```
