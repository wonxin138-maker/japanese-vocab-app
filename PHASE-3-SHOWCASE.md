### 猫咪结构 🐱

```
        左耳  右耳
         \  /
      左眼 眼 右眼
         \ | /
      ────●────  (鼻子)
          |       (嘴)
     脸颊○ ○脸颊
      
      ╪════════╪    (身体)
      ║        ║
      ║        ║
      ├────────┤
      │ ╱  ╲ │
      │ │  │ │     (前腿)
      │ │  │ │
      └─┴──┴─┘
      
        ╱╲╱╲╱╲     (尾巴)
        ─────
```

### 配色方案 🎨

| 部位 | 主色 | 配色 | RGB值 |
|------|------|------|-------|
| **身体** | 奶油色 | 淡橘色 | #F5DEB3 → #FFB3A1 |
| **头部** | 奶油白 | 淡橘色 | #FFF8F5 → #FFE8D6 |
| **腮红** | 粉红色 | 半透明 | #FFB3D9 (opacity: 0.5) |
| **眼睛** | 黑色 | 白光 | #1A1A1A + white |
| **鼻子** | 淡粉 | - | #FFB8A8 |

### 解剖学特征

**大头比例**
- 头部直径：130px
- 身体高度：120px
- 比例：约 1:1 （萌系风格）

**眼睛设计**
- 形状：椭圆形（宽25px × 高35px）
- 高光：右上角小圆点，营造"活泼感"
- 眨眼动画：3秒周期，5秒和15秒时闭合

**腮红优化**
- 形状：椭圆渐变
- 位置：眼睛下方两侧
- 效果：半透明粉红，营造"娇羞感"

**尾巴设计**
- 形状：S型曲线
- 长度：100px
- 摇晃幅度：-5° → 10° → 8°
- 周期：2.5秒

## 🎬 动画展示

### 动画1：呼吸动画（cat-breathe）

```
时间线：
  0%    → scale(1)      初始
  50%   → scale(1.03)   放大（吸气）
  100%  → scale(1)      缩小（呼气）

周期：3秒
效果：整个宠物缓慢放大→缩小，营造"活着"的感觉
播放：循环，无限
```

**视觉效果**：
```
缩小状态   →   放大状态   →   缩小状态
   🐱     →      😸      →      🐱
```

### 动画2：眨眼动画（cat-blink）

```
时间线：
  0-10%  → height: 35px   睁眼
  5%     → height: 4px    快速闭眼
  10-20% → height: 35px   睁眼
  15%    → height: 4px    快速闭眼
  20-100% → height: 35px  睁眼

周期：3秒
效果：眼睛定期闭合，营造"眨眼"感觉
播放：循环，无限
```

**眼睛状态**：
```
睁眼：●●
闭眼：──
睁眼：●●
```

### 动画3：尾巴摇晃（cat-tail-wave）

```
时间线：
  0%   → rotate(-5°)    向左
  25%  → rotate(10°)    向右（幅度大）
  50%  → rotate(-5°)    向左
  75%  → rotate(8°)     向右（幅度小）
  100% → rotate(-5°)    向左

周期：2.5秒
效果：S型自然摇晃，像真实猫咪摆尾巴
播放：循环，无限
```

**尾巴位置**：
```
向左    向右      向左    向右
  \     /          \     /
   \   /            \   /
    \ /              \ /
     V                V
```

### 动画4：腿部摇晃（cat-leg-sway）

```
时间线：
  0%, 100% → rotate(0deg)    正常
  50%      → rotate(-2deg)   微微左倾

周期：2秒
延迟：左腿 -0.2s，造成"交替走路"感
效果：两条腿轻微摇晃，营造"站立"感觉
播放：循环，无限
```

### 动画5：弹跳（cat-bounce）- 压力模式下

```
时间线：
  0%, 100% → translateY(0) scale(1)     地面
  50%      → translateY(-20px) scale(1.05)  空中

周期：2秒
触发：用户有压力时点击宠物
效果：温柔的上下浮动，表示"温暖陪伴"
播放：单次或循环
```

### 动画6：摇晃（shake）- 普通点击

```
时间线：
  多次从左到右的微小旋转

周期：300ms
触发：用户无压力时点击宠物
效果：快速左右摇晃，表示"欢快"
播放：单次
```

## 🎯 使用场景

### 场景1：用户心情不好

```
时间：晚上20:00
用户：最近心情不太好，工作压力很大

系统流程：
1. 检测"心情不好"(sadness类) + "压力"(stressed)
2. 压力级别判断：2级
3. 回复选择：sadness > general 类
4. 显示："你想哭的话，我听着…💙"
5. 表情气泡：蓝色➕粉色混合 🤍
6. 动画：呼吸效果，表达"陪伴"
```

### 场景2：用户工作繁忙

```
时间：白天工作时间
用户：我要准备3个报告，还要开5个会...受不了了

系统流程：
1. 检测多个压力词："受不了"(3)
2. 压力级别：3级（严重）
3. 立即激活overwhelm模式
4. 显示："一个人在承受的时候，告诉我好吗"
5. 表情：🤍（最温暖的）

```

```
用户行为：
1. 输入："我好累"
2. 宠物显示安慰
3. 用户点击宠物
4. 宠物继续显示相关安慰+呼吸动画

效果：
- 用户感到"宠物在陪伴"
- 重复点击能获得不同安慰
- 呼吸动画创造"活泼"感觉
```

用户心路：

  → 检测：无压力词
  → 宠物：普通对话

  → 检测最近消息：无压力

## 📊 压力检测矩阵

### 优先级判断

```
┌─────────────────────────────────────┐
│ 用户输入：分析→判断→响应            │
├─────────────────────────────────────┤
│                                     │
│ 第1层：问候/告别检查               │
│   ├─ 是 → 返回问候/告别             │
│   └─ 否 → 下一层                    │
│                                     │
│ 第2层：【NEW】压力检查             │
│   ├─ 压力级别 > 0 → 安慰模式       │
│   └─ 否 → 下一层                    │
│                                     │
│ 第3层：其他情感检查                │
│   ├─ love → 爱意回复               │
│   ├─ angry → 道歉回复               │
│   ├─ happy → 鼓励回复               │
│   ├─ sad → 同情回复                │
│   └─ 否 → 下一层                    │
│                                     │
│ 第4层：问题判断 / 状态检查         │
│                                     │
│ 第5层：默认聊天回复                │
│                                     │
└─────────────────────────────────────┘
```

### 关键词映射表

```javascript
压力关键词库：
├─ 疲劳类：'累', '困', '疲劳', ...
├─ 烦躁类：'烦', '烦人', '受不了', ...
├─ 悲伤类：'难过', '伤心', '哭', ...
├─ 压力类：'压力', '崩溃', '撑不住', ...
└─ 强度修饰：多个关键词 → 更高压力级别

对应回复库：
├─ tiredness: 6条疲劳专用回复
├─ overwhelm: 5条压力专用回复
├─ sadness: 5条悲伤专用回复
└─ general: 4条通用回复
```

## 🔧 技术实现细节

### CSS渐变公式

```css
/* 3D立体感 = 渐变 + 内阴影 */
body {
  background: linear-gradient(135deg, #FFF5EB, #FFE8D6);
  box-shadow: 
    inset -2px 2px 5px rgba(0,0,0,0.05),   /* 左上暗 */
    0 8px 20px rgba(255,166,216,0.25);     /* 底部投影 */
}
```

### 压力检测算法

```javascript
_detectStress(text) {
  const keywords = PET_DATA.emotionalKeywords.stressed;
  let count = 0;
  
  for (const keyword of keywords) {
    if (text.includes(keyword)) count++;
  }
  
  // 关键词计数到压力级别的映射
  return Math.min(count, 3);  // 最多3级
}
```

### 动画触发机制

```javascript
// 点击宠物时
_handlePetClick() {
  const stressLevel = chatAI._detectStress(lastMessage);
  
  if (stressLevel > 0) {
    // 压力模式：温柔动画
    pet.style.animation = 'cat-breathe 1.5s ease-in-out';
  } else {
    // 普通模式：活泼动画
    pet.classList.add('shake');
  }
}
```
