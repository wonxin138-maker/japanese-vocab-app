## 🔍 问题排查

### 问题1：3D猫咪不显示

**检查步骤**：
```javascript
// 打开 F12 开发者工具
// 在 Console 输入：
petCat3D
// 应该返回一个 PetCat3D 对象
```

**解决**：
- [ ] 是否等待页面完全加载？
- [ ] 浏览器是否支持WebGL？
  ```
  在新标签页打开：
  chrome://gpu 或
  about:debugging → 查看GPU是否启用
  ```
- [ ] 是否有JavaScript错误？
  按 F12 查看 Console 标签

### 问题2：加载很慢

**原因**：Three.js库较大（~600KB）

**解决**：
- [ ] 检查网络速度
- [ ] 清除浏览器缓存（Ctrl+Shift+Delete）
- [ ] 首次加载会较慢，刷新重试

### 问题3：性能不流畅（卡顿）

**检查**：
```
F12 → Performance → 记录 10秒
查看是否有长任务（>16ms）
```

**解决**：
- [ ] 关闭其他浏览器标签页
- [ ] 更新显卡驱动
- [ ] 尝试其他浏览器

### 问题4：对话不显示

**检查**：
- [ ] 输入框是否聚焦？
- [ ] 是否按了 Enter 或点击发送按钮？
- [ ] JavaScript 是否启用？
