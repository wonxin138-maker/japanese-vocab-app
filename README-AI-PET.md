# 🐾 AI宠物 - 你的治愈伙伴

> 一个生活在屏幕中的AI宠物，可以陪伴你、聊天、表达情绪。这是一个完整的前端项目，展现了情感AI、状态管理、动画设计的完美结合。

## ✨ 核心特色

### 🤖 智能AI对话
- 自然语言理解（感知用户情绪）
- 9种情感识别系统
- 主动开启话题（定时触发）
- 真实的长期未互动反馈

### 🎭 丰富的情感表达
- 心情值、饥饿值、精力值三维状态
- 根据状态动态调整对话风格
- 自然的衰减和恢复机制
- 状态可视化展示

### 💬 有趣的互动方式
- 自由聊天（输入框对话）
- 点击宠物（触发动作和反馈）
- 喂食系统（减少饥饿）
- 陪玩系统（增加心情）
- 随机惊喜（表情包、金句、笑话）

### 🎨 治愈系视觉设计
- 极简 + 韩系 + 柔和配色
- 流畅的CSS动画
- 完整的响应式设计
- 可爱的粉色宠物形象

### 💾 智能数据管理
- 自动保存（无需手动）
- 浏览器本地存储（隐私保护）
- 可恢复的对话历史
- 数据导出功能

## 🎯 系统设计亮点

### 架构设计
```
┌─ 表现层 (UI)
│  ├─ HTML结构
│  ├─ CSS动画
│  └─ 实时通知
├─ 业务逻辑层
│  ├─ 状态管理 (State)
│  ├─ 对话引擎 (Chat AI)
│  └─ 事件处理
└─ 数据持久化层
   └─ LocalStorage
```

### 核心特性

1️⃣ **情感识别**
   - 关键词提取 → 情感分类 → 回复选择
   - 支持爱、喜、怒、哀、惧5种基本情感

2️⃣ **状态系统**
   - 三维指标：心情 + 饥饿 + 精力
   - 自动衰减 + 用户互动补偿
   - 状态影响对话风格

3️⃣ **主动对话**
   - 每5-20分钟随机触发
   - 根据宠物状态智能选择话题
   - 营造"生命感"

4️⃣ **数据持久化**
   - 自动保存（30秒一次）
   - 浏览器重启保留数据
   - 支持手动导出

## 🔧 自定义修改

### 修改宠物名字

编辑 `pet-data.js` 第3行：
```javascript
name: '毛毛',  // 改成你喜欢的名字
```

### 修改对话

编辑 `pet-data.js` 中的 `responses` 对象：
```javascript
greeting: [
    '你的第一条对话',
    '你的第二条对话',
    // 添加更多...
]
```

### 修改颜色

编辑 `styles.css` 第8-11行：
```css
:root {
    --primary-color: #FFD6E8;    /* 改你想要的颜色 */
    --accent-color: #FFA8D8;
}
```

### 修改宠物形象

编辑 `pet.html` 中的SVG元素：
```html
<circle cx="100" cy="120" r="45" fill="#FFD6E8"/>
<!-- 改这里的数值和颜色 -->
```

## 💡 常见问题

### Q: 可以在手机上用吗？
**A:** ✅ 完全支持！响应式设计，竖横屏自适应。

### Q: 数据会丢失吗？
**A:** ❌ 不会！数据保存在浏览器本地，关闭页面也保留。

### Q: 需要网络吗？
**A:** ❌ 不需要！完全离线工作（除非使用API）。

### Q: 能导出数据吗？
**A:** ✅ 能！在控制台运行 `copy(app.getInfo())` 即可复制。

### Q: 怎样重置宠物？
**A:** 在控制台运行 `app.reset()` 或 `localStorage.clear()`。

### Q: 可以改成自己的AI吗？
**A:** ✅ 可以！详见 ADVANCED-FEATURES.md 的 OpenAI集成部分。

**更多FAQ → [快速开始文档](QUICK-START-PET.md)**
